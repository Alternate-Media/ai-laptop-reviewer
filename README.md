# Laptop Reviews Site

Technical, "Anti-Hype" laptop reviews for professionals and creatives.

## Tech Stack
- **SSG**: [Hugo](https://gohugo.io/)
- **CSS**: [Bootstrap 5](https://getbootstrap.com/)
- **Management**: [mise](https://mise.jdx.dev/)

## Operations
This site uses the **Hugo Operations** agent for content lifecycle and maintenance.
